# Dikshit Neupane - Personal Portfolio & Interactive Terminal

A highly polished, modern, and interactive developer portfolio site built using vanilla HTML5, CSS3, and JavaScript. Designed with a cyber-tech aesthetic, subtle glassmorphism, floating glow effects, and a fully functional terminal emulator that mimics a Linux CLI shell.

## 🚀 How to Run the Website

### Option 1: Direct Execution (Simplest)
1. Navigate to the `portfolio` folder.
2. Double-click the [index.html](file:///C:/Users/npneu/portfolio/index.html) file to open it in your preferred web browser.

### Option 2: Live Server via Python
If you want to run it via a local development server:
1. Open your terminal (PowerShell, Bash, CMD).
2. Navigate to this directory:
   ```bash
   cd "C:\Users\npneu\portfolio"
   ```
3. Run the Python HTTP server:
   ```bash
   python -m http.server 8000
   ```
4. Open your browser and go to `http://localhost:8000`.

### Option 3: VS Code Live Server extension
If you use VS Code, you can open the project folder and click **Go Live** on the bottom status bar.

## 📂 Project Structure

- **[index.html](file:///C:/Users/npneu/portfolio/index.html)**: The core semantic HTML5 structure of the website. Includes sections for Hero, About, Skills, Terminal, Projects, and Contact details.
- **[style.css](file:///C:/Users/npneu/portfolio/style.css)**: Custom styling sheet. Features floating background orbs, digital grid overlays, glassmorphic container elements, responsive media queries, and dynamic hover scales.
- **[script.js](file:///C:/Users/npneu/portfolio/script.js)**: Custom interactivity script. Handles active navbar styling, scroll trigger transitions, mobile navigation drawer opening, and the full logic of the interactive shell.
- **`assets/`**: Directory containing assets.
  - **`profile.jpg`**: Your personal profile picture copied from your OneDrive screenshots folder.

## 💻 Terminal Emulator Commands
In the **Interactive Shell** section of your website, try typing the following commands:
* `help` - Lists all available commands.
* `about` - Displays your official biography description.
* `skills` - Lists your tech stack categorizations.
* `projects` - Shows lists of featured code repositories.
* `contact` - Shows active links to your social profiles and email.
* `neofetch` - Shows a customized Linux neofetch card.
* `matrix` - Toggles an animated digital rain effect falling in the background of the shell terminal!
* `clear` - Wipes the shell console line history.
